# 密匙
sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-keys E298A3A825C0D65DFD57CBB651716619E084DAB9
sudo apt-get update
sudo apt-get install r-base r-base-dev

  • 虚拟机安装ubuntu后切换工作区快捷键ctrl + alt + 方向键，上下方向键可正常切换，而左右方向键切换不了

    因为虚拟机中ctrl + alt + 左右方向键在VMware中用于切换全屏虚拟机，修改VMware切换全屏 虚拟机快捷键即可，如ctrl + shit + 左右方向键
